# 语音通话模式设计

## 概述

为 LocalChat Web 界面增加"语音通话"模式——用户按住按钮说话、松开发送，AI 以语音回复，全程无需打字或看文字，像打电话一样自然。

## 设计目标

- 纯语音交互：语音模式下手打文字输入框隐藏，只保留通话按钮
- 消息气泡：用户和 AI 的消息均以"语音气泡"展示，可点击播放/暂停，显示时长
- 自动流转：用户录音 → STT（不可见）→ 对话 API（不可见）→ TTS → 自动播放 AI 语音
- 无缝切换：与现有文字聊天模式互不干扰，一键切换

## 交互流程

```
用户按住 🎤 按钮 → 开始录音
用户松手 → 停止录音 / 上传 / STT
  → 语音气泡（用户）插入对话
  → /api/chat（不可见，无文字流）
  → TTS → 语音气泡（AI）插入对话 → 自动播放
```

## UI 方案

### 头部切换开关
- 在 header 区域增加一个切换按钮，现有文字模式 ↔ 语音通话模式
- 语音模式下输入区域改为 Push-to-Talk 按钮，文本输入框隐藏
- 切换回文字模式时，正常显示文本框

### 语音气泡设计
- 用户语音气泡：右侧，🎤 图标 + 播放按钮 + 时长（与微信语音气泡样式类似）
- AI 语音气泡：左侧，角色头像 + 播放按钮 + 时长 + 声波动画
- 点击播放/暂停，播放时进度条/声波动画
- 点击其他气泡时自动停止当前播放

### 后端变更

#### `/api/chat` 扩展
- 新增参数 `mode: "voice"`（可选）
- `mode: "voice"` 时 SSE 不返回 text delta，只在完成后返回一条含 TTS 种子等元数据的 JSON
- 也可在现有 SSE 流结束后，由前端自动触发 `/api/tts`

**采用方案：SSE 流不变 + 前端自动 TTS**
- `/api/chat` 不做改动，正常返回 SSE text delta
- 前端收集完整 AI 回复文本后，自动调用 `/api/tts`
- 优点：改动最小，复用现有逻辑

### 前端变更

| 模块 | 变更内容 |
|------|----------|
| `index.html` | 新增 `voiceMode` 状态变量 |
| Header | 新增语音模式切换按钮 |
| 输入区 | voice-only 时隐藏 textarea + send，显示 push-to-talk 按钮 |
| 消息区 | 新增语音气泡渲染函数 |
| 录音 | 复用现有 MediaRecorder 逻辑 |
| TTS 播放 | 复用现有 Audio API + 自动播放逻辑 |

### 关键技术细节

1. **Push-to-talk**: `mousedown` 开始录音，`mouseup`/`mouseleave` 停止并上传
2. **STT 结果**: 上传后调用 `/api/stt` 获取文本，将文本作为用户消息插入对话（但不显示文本）
3. **AI 回复**: 正常调用 `/api/chat`（SSE 流），前端在后台收集完整文本
4. **TTS 自动播放**: SSE 结束后自动触发 `/api/tts`，播放完成后标记气泡状态

### 语音消息数据模型

```javascript
// 语音气泡消息体
{
  type: "voice",
  role: "user" | "assistant",
  text: "...",       // STT 识别的文本（后台用，气泡上不显示）
  audioUrl: "...",   // Blob URL 或 /api/tts 返回的音频
  duration: 3.2,     // 秒
  isPlaying: false
}
```

### 约束与边界情况

- 浏览器需支持 `MediaRecorder` API（现代浏览器均支持）
- 录音需用户授权麦克风
- TTS 自动播放需浏览器 `autoplay` 策略支持（用户已与页面交互过即可）
- 录音时长上限：60 秒（超过自动停止并发送）
- 播放中发送新语音 → 自动停止当前播放

## 实施计划

1. **HTML/CSS**: 添加语音模式切换按钮、push-to-talk 按钮、语音气泡样式
2. **JS 状态管理**: 添加 `voiceMode` 状态、切换逻辑
3. **录音交互**: push-to-talk 录音逻辑（mousedown/mouseup）
4. **语音气泡渲染**: voice-only 消息渲染、播放/暂停控制
5. **自动 TTS 播放**: SSE 完成后自动触发 TTS 并播放
6. **集成测试**: 全流程联调
