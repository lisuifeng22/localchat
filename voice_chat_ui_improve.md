## 语音通话 UI 改进计划（2026-05-17）

当前语音通话 UI 粗糙，已规划以下改进方向：

1. **录音时实时音量条** — 用 AnalyserNode 取麦克风音量，在 PTT 按钮周围画动态圆环
2. **动画过渡** — 文字/语音模式切换时渐变过渡
3. **状态指示器** — 输入区上方显示 "🎤 录音中..." / "⏳ 识别中..." / "🤖 AI 回复中..." / "🔊 播放中..."
4. **波形优化** — 气泡波形改成实际音频的频谱预览
5. **播放进度条** — AI 语音播放时显示进度条，支持拖动
6. **取消录音** — 手指滑出按钮区域自动取消录音

**优先级：** 待定，等用户确认范围后按 brainstorming 流程进入设计阶段。
